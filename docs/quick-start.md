## 📊 评级标准

| 评级 | 互动率 | 平均点赞 | 推荐度 | 预算建议 |
|------|--------|---------|--------|----------|
| ⭐⭐⭐⭐⭐ | > 50% | > 10,000 | 强烈推荐 | 8-15万 |

- **互动率** = 平均点赞 / 粉丝数 × 100%
- **互动指数** = 点赞 + 评论×10 + 分享×5 + 收藏×3

## ⚠️ 常见问题

### Q1: 环境检查失败怎么办？

**A:** 按照检查脚本的提示逐项修复：
- 登录缓存不存在 → 重新登录
- 残留进程 → 清理进程
- 目录不存在 → 创建目录

### Q2: 搜索结果太少怎么办？

**A:**
1. 修改关键词（更通用的词）
2. 调整粉丝数范围（降低下限）
3. 多次搜索不同关键词

### Q3: 深度抓取失败怎么办？

**A:**
1. 检查登录状态
2. 清理残留进程
3. 降低并发数（一次只抓 2-3 个达人）
4. 等待 5-10 分钟后重试

### Q4: 分析结果全是低评级怎么办？

**A:**
1. 检查是否是数据问题（粉丝数为 0）
2. 调整评级标准（降低阈值）
3. 继续搜索其他关键词

### Q5: Token 消耗太高怎么办？

**A:**
1. ✅ 使用 `analyze_creators.py` 脚本（减少 70% token）
2. ✅ 使用标准化 Prompt 模板
3. ✅ 批量处理，不要单个分析
4. ✅ 避免重复生成报告
