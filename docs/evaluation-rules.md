**更新时间**：2026-03-04
**脚本**：`process_mediacrawler_data_v2.py`

---

## 📊 评级体系（S/A/B/C/D）

| 评级 | 条件 | 说明 | 投放建议 |
|------|------|------|---------|
| **S级** | 粉赞比>5% + 平均点赞>1000 + 样本≥5条 + 稳定性>30% | 优质且稳定 | ✅ 强烈推荐，可直接投放 |
| **A级** | 粉赞比>5% + 平均点赞>1000，但样本<5条或不稳定 | 优质但有风险 | ⚠️ 高风险高回报，根据风险偏好选择 |
| **B级** | 粉赞比2-5% | 良好 | 🧪 可小额测试 |
| **C级** | 粉赞比1-2% | 一般 | ⚠️ 需谨慎 |
| **D级** | 粉赞比<1% | 差 | ❌ 不建议投放 |

---

## 🎯 核心指标

### 1. 粉赞比
```
粉赞比(%) = (平均点赞 / 粉丝数) × 100
```

**参考基准**：
- >5%：优质
- 2-5%：良好
- 1-2%：一般
- <1%：差

### 2. 稳定性
```
稳定性 = 最低点赞 / 平均点赞
```

**阈值**：
- >30%：稳定，风险低
- <30%：不稳定，可能遇到低谷期

**重要性**：避免投放时遇到低谷期导致ROI爆炸

**示例**：
- 兔哥来报：平均5372赞，最低76赞，稳定性1.4% ❌
- 如果投放费用几千块，遇到低谷期只有76赞，ROI灾难

### 3. 样本量
```
样本量 = 实际抓取的视频数（2-5条）
```

**影响**：
- ≥5条：数据充分
- 2-4条：新号，样本不足，需观察

### 4. 趋势
```
趋势 = 对比最近2条和最早2条视频的平均点赞
```

**判断**：
- 上升：近期表现更好
- 下降：近期表现变差
- 波动：数据不稳定
- 样本不足：视频<3条

---

## 🏷️ 风险标签

### 新号(N条)
- **条件**：视频数<5条
- **风险**：样本不足，数据不够充分
- **建议**：可尝试，但需持续观察

### 不稳定(最低X赞)
- **条件**：稳定性<30%
- **风险**：投放可能遇到低谷期
- **建议**：
  - 小粉丝达人（<1万）：可尝试，投放成本低
  - 大粉丝达人（>1万）：风险大，谨慎投放

### 下降趋势
- **条件**：近期表现明显变差
- **风险**：可能正在掉粉或内容质量下降
- **建议**：谨慎投放，或等待恢复

## 📋 分类规则

### 优质达人.csv
**收录标准**：S级 + A级

- **S级**：可直接投放
- **A级**：根据以下因素判断是否投放：
  - 粉丝数（小粉丝风险低）
  - 风险标签（新号还是不稳定）
  - 预算（低预算可尝试）

### 备选达人.csv
**收录标准**：B/C/D级

- **B级**：可小额测试
- **C级**：需谨慎评估
- **D级**：建议放弃

## 🔍 实际案例

### 案例1：S级达人（理想情况）
- 粉丝：10,000
- 5条视频点赞：800, 900, 1000, 1100, 1200
- 平均点赞：1000
- 粉赞比：10%
- 稳定性：80/1000 = 80% ✅
- **评级**：S级
- **建议**：强烈推荐，数据稳定

### 案例2：A级达人（不稳定）
- 粉丝：7,619
- 5条视频点赞：76, 732, 10231, 15274, 551
- 平均点赞：5372
- 粉赞比：70%
- 稳定性：76/5372 = 1.4% ❌
- **评级**：A级
- **风险标签**：不稳定(最低76赞), 下降趋势
- **建议**：高风险高回报，谨慎投放

### 案例3：A级达人（新号）
- 粉丝：99
- 3条视频点赞：4, 1662, 87
- 平均点赞：584
- 粉赞比：590%
- 稳定性：4/584 = 0.7% ❌
- 样本量：3条
- **评级**：B级（粉赞比不满足A级要求）
- **风险标签**：新号(3条)
- **建议**：新号高潜力，可小额测试

### 案例4：B级达人（稳定）
- 粉丝：15,610
- 5条视频点赞：3481, 137, 99, 168, 116
- 平均点赞：800
- 粉赞比：5.13%
- 稳定性：99/800 = 12% ❌
- **评级**：B级
- **建议**：可小额测试

---

## 📌 记住

1. **稳定性 > 粉赞比**：数据不稳定的达人风险大
2. **不跳过新号**：新号可能是高潜力达人
3. **风险标签很重要**：根据标签判断投放风险
4. **分级是参考**：最终决策需结合预算和风险偏好
5. **A级不等于差**：A级是高潜力但有风险，可以尝试
